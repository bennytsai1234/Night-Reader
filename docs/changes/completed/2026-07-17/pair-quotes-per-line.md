# pairQuotes 逐行配對（打磨方向 3）

層級：T1（transformer 純函式層，可逆）

## Before

直引號激進項 `pairQuotes`（`"` 交替配對 → 「」）以**整章**為配對單位：
全章引號總數為奇數就整章放棄；即使偶數，一個雜訊引號也會讓其後
所有開/收全部顛倒。長篇章節命中率極低、錯位風險高——這是它只能
預設關的主因之一。

## After

配對改以**行**為單位（對白引號幾乎不跨行）：奇數個引號的行原樣
保留，單一雜訊引號的影響被隔離在該行，其他行正常配對。跳脫引號
（`\"`）判斷不變。另補激進項誤傷對照表測試：collapse 不吃 `……`、
removeCjkSpaces 保留西文詞間空格。

預設值維持關閉；可靠度數據累積足夠後再個別評估（彎引號 “” 已在
歧義寬度標點正規化中預設轉「」，直引號書源殘量待真書觀察）。

驗證：`flutter analyze` 無問題；`flutter test` 743 全過（新增 2 組
對照表測試）。
