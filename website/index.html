探索功能

功能總覽

為深度閱讀而生

從排版引擎到語音朗讀，每個細節都為長篇中文閱讀細心打磨。

精細排版引擎

字號、行高、字距、段距、首行縮排、左右邊距皆可獨立調整。針對中文長篇閱讀最佳化，標點符號避頭尾，版面乾淨整齊。

雙翻頁模式

支援「平移翻頁」（左右滑動）與「上下滾動」（連續捲動）兩種閱讀模式，自由切換，隨心閱讀。

語音朗讀 TTS

逐詞高亮、續讀、自動接章。可調整語速、音調、音量與音色，選擇不同的語音引擎。

多書源並行搜尋

在所有已匯入的來源中同時搜尋，快速找到想看的內容。支援探索分類瀏覽。

書源管理

網路／本地匯入、匯出、批次有效性校驗、訂閱自動更新、規則偵錯工具，一應俱全。

書架與書籤

書架統一管理在讀書籍，顯示封面、閱讀進度與快取狀態。書籤標記段落，隨時跳回。

離線快取

手動預下載章節以供離線閱讀。背景下載佇列可查看、暫停、重試與刪除任務。

一鍵備份還原

書架、書源、替換規則、書籤、閱讀進度與 App 設定一次備份，換機或重裝後完整還原。

本地電子書

支援匯入並閱讀 TXT 與 EPUB 格式的本地檔案，無需網路即可閱讀。

自訂主題外觀

切換閱讀背景、字體與配色。點擊區域自訂（打點區），貼合個人操作習慣。

替換規則

支援全域與章內替換規則，自訂淨化或修正內文，打造純淨閱讀體驗。

繁簡轉換

可選擇不轉換、簡轉繁或繁轉簡，滿足不同語言習慣與閱讀偏好。

技術架構

設計與工程

以 Flutter 為基底，自製排版渲染引擎與通用網頁規則解析器。

Flutter Dart Drift (SQLite) Dio 網路層 QuickJS 引擎 HTML / CSS / XPath 解析 自製排版渲染引擎 Provider 狀態管理 flutter_tts WorkManager 背景任務
