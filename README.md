## 簡介

夜讀（Night Reader）是一款以長篇小說閱讀為核心的 Android 閱讀工具。它**不內建任何書籍內容**，而是提供一套通用的網頁規則解析引擎與本地文本排版引擎：使用者匯入自己信任且具合法授權的書源後，即可在單一介面中完成搜尋、閱讀、語音朗讀、離線快取與資料備份。

App 採用自製的排版與渲染引擎，針對中文長篇閱讀最佳化，並將書源、書架、設定與閱讀進度完整保存在本機，可隨時備份遷移。

> 夜讀為純工具型應用，本身不提供、不託管、不分發任何書籍或書源。請參閱文末[免責聲明](#免責聲明)。

## 功能詳述

### 書源管理
- **多種匯入方式**：支援網路 URL 匯入與本地檔案匯入。
- **批次有效性校驗**：一鍵在背景（isolate）批次驗證書源，自動標示連線失敗、需登入或需付費訂閱才能存取的來源。校驗僅檢查規則格式與基本連線狀態，**通過校驗不代表內容已獲合法授權**。
- **訂閱自動更新**：可設定訂閱來源，讓書源規則隨上游更新。
- **匯出與偵錯**：支援書源匯出分享，並內建規則偵錯工具，協助定位抓取問題。

### 閱讀體驗
- **精細排版**：字號、行高、字距、段距、首行縮排、左右邊距皆可獨立調整。
- **雙翻頁模式**：支援「平移翻頁」（左右滑動）與「上下滾動」（連續捲動）兩種模式。
- **自動翻頁**：可開啟自動翻頁並調整速度，解放雙手。
- **點擊區域自訂（打點區）**：自訂螢幕各區塊的點擊行為，貼合個人操作習慣。
- **中文排版最佳化**：標點符號避頭尾，版面乾淨整齊。
- **主題與外觀**：切換閱讀背景、字體與配色。

### 語音朗讀（TTS）
- 朗讀時**逐區塊高亮**，清楚標示目前進度。
- 暫停後**從原處續讀**，一章讀畢**自動接續下一章**。
- 可調整**語速、音調、音量、音色**，並選擇系統語音引擎。

### 搜尋與探索
- **多書源並行搜尋**：在所有已匯入且確認具合法授權的來源中同時搜尋。
- **探索分類**：瀏覽書源提供的探索／分類頁，發現新內容。

### 書架與書籍管理
- 書架統一管理在讀書籍，顯示封面、閱讀進度與已快取章節數。
- 書籍詳情頁可管理目錄、封面與閱讀紀錄。
- **書籤**：標記重要段落，隨時跳回。
- **替換規則**：支援全域與章內替換規則，自訂淨化或修正內文。

### 離線快取
- 加入書架僅儲存書籍資訊、目錄與閱讀進度，**不自動下載正文**。
- 對於確認具合法授權的內容，可在書籍詳情頁手動**預下載章節**以供離線閱讀。
- **背景下載佇列**可查看、暫停、重試與刪除下載任務，並提供快取清理。

### 資料備份與遷移
- **一鍵備份還原**：書架、書源、替換規則、書籤、閱讀進度與 App 設定可一次備份，換機或重裝後完整還原。
- **書架交換**：匯出夜讀書架以便分享或遷移。
- **繁簡轉換**：可選擇不轉換、簡轉繁或繁轉簡。

### 本地電子書
- 支援匯入並閱讀 `TXT` 與 `EPUB` 格式的本地檔案。

## 快速上手

1. 開啟 App，進入**書源管理**。
2. 點擊右上角選單，選擇**網路匯入**或**本地匯入**，加入您信任且具合法授權的書源。
3. 於**書源管理**頁執行批次校驗，篩除無效來源。
4. 前往**搜尋**頁，在已匯入的來源中搜尋內容並加入書架。
5. 回到書架點擊書籍，即可開始閱讀；如需離線，於書籍詳情頁手動預下載章節。

## 常見問題

**Q：安裝後沒有任何小說？**
夜讀是純工具型閱讀器，不提供也不儲存任何書籍內容。請先匯入書源。

**Q：支援哪些本地電子書格式？**
支援 `TXT` 與 `EPUB`。

**Q：需要登入或驗證碼的書源能用嗎？**
需要瀏覽器互動驗證（如登入、驗證碼）的書源目前不支援，此類書源會直接回報錯誤，建議改用不需驗證的來源。

**Q：加入書架後會自動下載整本書嗎？**
不會。加入書架僅保存書籍資訊與閱讀進度，正文需手動預下載。

**Q：換手機後資料會不見嗎？**
不會。使用備份還原功能即可完整遷移書架、書源、書籤與設定。
