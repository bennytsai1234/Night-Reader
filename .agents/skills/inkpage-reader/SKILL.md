# 墨頁 Inkpage Reader Codebase Atlas

這是 Codebase Atlas 在 Codex 的入口技能。

## 使用方式

1. 保留使用者的原始請求。
2. 開啟 `../../../docs/inkpage_reader_main_workflow.md` 並遵循工作流程。
3. 在任何操作開始之前，先閱讀 atlas 索引。
4. 對任何涉及編輯檔案的操作，提供「之前 / 之後」並等待使用者確認後再編輯。
5. 依此交付策略完成：建立 commit 並 push。

## 禁止事項

- 不要在使用者明確要求完整重建時才重新執行 Codebase Atlas 初始化。
- 不要略過閱讀 atlas 索引。
- 不要在使用者確認「之前 / 之後」之前編輯檔案。
